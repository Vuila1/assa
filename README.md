AI Aimbot

Scaffolded aimbot project: Win32 mouse adapter, aimbot movement controller, shooting thread, simple overlay and control panel.

Run:
- Create a virtual environment and install dependencies:
  python -m venv .venv
  .venv\Scripts\activate
  pip install -r requirements.txt

- Run the control panel (recommended):
  python -m AI.main

Notes:
- Windows only (SendInput). Use responsibly; automated input may be detected by anti-cheat.
