aimbot_project
===============

Restructured layout for the aimbot project. Top-level packages:

- core: Core engine and logic
- gui: User interface layer (CLI/GUI)
- config: Default configuration and loaders
- models: Domain models used by the engine
- utils: Helper utilities
- data: Directory for datasets and runtime data

Usage:
- python -m aimbot_project.main [config.json]

Fill in implementation details for your project-specific logic (vision, input, models, etc.).
